NeoVim Setup on Windows

This is brief guide to setting up nvim with plugins (using Plug) on Windows. I assume basic familiarity with vim, including using command mode.

  • Download NeoVim and unpack it to a directory (manually, or with scoop install neovim).
  • Open nvim in your terminal or open nvim-qt. This will be in the bin sub-directory.
  • Find the config directory using :echo stdpath('config'). I will assume this is the result from now on: C:\Users\Name\AppData\Local\nvim. You will need to replace this with yours for the following instructions.
  • Open powershell in that folder, and run the following:
New-Item init.vim -ItemType file
New-Item autoload -ItemType directory
New-Item plugged -ItemType directory

# Download plug.vim
$uri = 'https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim'
Invoke-WebRequest -Uri $uri -OutFile "autoload\plug.vim"
  • Add plugins to init.vim by pasting in the below. Search online to learn more about how to use each plug-in. Remove any that you don’t want.
call plug#begin('C:/Users/Name/AppData/Local/nvim/plugged')

Plug 'morhetz/gruvbox'    " theme
Plug 'neoclide/coc.nvim'
Plug 'machakann/vim-highlightedyank'
Plug 'tpope/vim-surround'
Plug 'scrooloose/nerdtree', { 'on':  'NERDTreeToggle' }

Plug 'junegunn/fzf.vim'
Plug 'junegunn/fzf', { 'do': { -> fzf#install() } }

Plug 'vim-airline/vim-airline'

call plug#end()
  • Close and re-open nvim and run :PlugInstall. Wait until the installation finishes.

And that’s it.

Bonus content: if you want to save and reload your vim session settings, you’ll need to edit the init.vim file. For example, you can append the following to init.vim:

  colorscheme gruvbox " Set color scheme
  set number          " Line numbers
  set mouse=a         " Automatically enable mouse usage
  nnoremap Y y$       " For consistency with C, D
